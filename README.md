# Usage
<!-- usage -->
```sh-session
$ npm install -g occ-tools-cli
$ occ-tools-cli COMMAND
running command...
$ occ-tools-cli (-v|--version|version)
occ-tools-cli/0.0.0-development linux-x64 node-v14.5.0
$ occ-tools-cli --help [COMMAND]
USAGE
  $ occ-tools-cli COMMAND
...
```
<!-- usagestop -->
```sh-session
$ npm install -g occ-cli
$ occ-cli COMMAND
running command...
$ occ-cli (-v|--version|version)
occ-cli/0.0.0 linux-x64 node-v14.3.0
$ occ-cli --help [COMMAND]
USAGE
  $ occ-cli COMMAND
...
```
